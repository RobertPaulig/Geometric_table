# Том I (name3.tex / name3.pdf) — конспект

Этот файл фиксирует ключевые идеи «Tom I» (QSG v5.0 / geom-spec v4.0) в текстовом виде, чтобы ИИ и snapshot могли прочитать базу без PDF. Полный оригинал — `docs/name3.pdf` (для людей) и `docs/name3.tex` (LaTeX-источник).

## 1. Онтология и постулаты
- Введены сущности: Среда (Medium), спектральное состояние, Закон роста дерева, геометрическая периодическая таблица.
- QSG трактует гравитацию как следствие спектрального распределения уровней; спектральные плотности $F_{\text{levels}}$ и геометрические функционалы $F_{\text{geom}}$ формируют инварианты.
- Антропный контур: AI-соавтор Antigravity выступает R&D-инженером, фиксированы требования “hypothesis > script > report”.

## 2. Фоновое пространство и спектральное состояние
- Задано фоновое пространство (ячеистая среда) и спектральный оператор $H$.
- Разделы о плотности состояний, функции Грина, солитонах и их роли для «материальных объектов» (узлы/графы).
- Определяются базы для $F_{\text{levels}}$, $F_{\text{geom}}$, $F_{\text{mass}}$.

## 3. Геометрическая периодическая таблица (H–Ne)
- Введён термин «геометрический атом»: граф вершина/порты/углы/ролей D/A.
- Мини-таблица H–Ne: черновики для углов связей, ролей элементов, электроотрицательностей (в том числе знаковой).
- Демонстрации на CH$_4$, NH$_3$, H$_2$O — проверка функционала на тестовых молекулах.

## 4. Функционалы и инварианты
- Общая структура функционала состояния: суммирование вкладов $F_{\text{levels}}$, $F_{\text{geom}}$, $F_{\text{mass}}$.
- Определены требования к “спектральной таблице”: дерево роста, collision rates, coverage и т.д.
- Сформулированы DoD для каждой гипотезы: наличие скрипта, результатов, записи в отчётах.

## 5. Ссылки на практические артефакты
- `analysis/geom/*`, `analysis/ws/*` реализуют сканеры и отчёты, описанные в тексте.
- Decision log (`docs/05_decision_log.md`) фиксирует, как идеи из Tom I переходят в R&D (Tom II / `docs/name4.tex`).
- Новый контур HETERO-1A (см. `docs/name4.tex` и `analysis/chem/*`) следует этим постулатам: tie-aware AUC, спектральные гейты и т.д.

> Для детальной математики, доказательств и рисунков используйте оригинал `docs/name3.pdf / name3.tex`. Настоящий файл — текстовое «зеркало» ключевых пунктов, чтобы snapshot был самодостаточен.
